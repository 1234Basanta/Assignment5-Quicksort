# Assignment 5: Quicksort Algorithm

## Overview

This project implements two versions of the Quicksort algorithm using Python:

- Deterministic Quicksort
- Randomized Quicksort

The project compares the performance of both algorithms on different types of datasets, including:

- Random data
- Sorted data
- Reverse-sorted data

## Files

- quicksort.py – Python implementation of both algorithms.
- Report.docx – Report describing the implementation, analysis, and results.
- README.md – Project documentation.

## Requirements

- Python 3.x

## How to Run

1. Open the project folder in Visual Studio Code.
2. Open quicksort.py.
3. Click the Run button or execute:

python quicksort.py

The program will display the execution times for both algorithms using different datasets.

## Summary

The randomized version of Quicksort provides more consistent performance because the pivot is selected randomly, reducing the likelihood of worst-case performance.
